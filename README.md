## 📝 Introduce
투자 다이어리 서비스 투다는 어떤 생각을 가지고 어떤 종목을 투자 했는지

어떤 뉴스 혹은 기사를 통해 종목을 얼마나 구입했는지를 쉽게 기록하는것을 도와주는 서비스 입니다.

현재 IOS 만 준비중에 있으며 2022년 1월 중 출시예정입니다 :)


## ⭐ Main Feature
- 투자 메모 기록 및 검색


## 🔧 Stack
**Backend, Devops**
- **Language** : Java
- **Library & Framework** : Spring Boot, Query-DSL
- **Database** : Mysql
- **ORM** : JPA
